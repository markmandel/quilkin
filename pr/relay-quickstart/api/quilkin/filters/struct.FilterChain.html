Struct quilkin::filters::FilterChain

source · 
pub struct FilterChain { /* private fields */ }
Expand description

A chain of Filters to be executed in order.

Executes each filter, passing the ReadContext and WriteContext between each filter’s execution, returning the result of data that has gone through all of the filters in the chain. If any of the filters in the chain return None, then the chain is broken, and None is returned.
